# How to use

-  Make sure to backup your .vimrc
-  Copy my `.vimrc` file to your `home` folder
-  Set up Vundle: run `git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim`
-  Start Vim and run `:PluginInstall`
-  Restart Vim. 
-  Profit!
